## Import data

Available formats:

- **csv**: various separators, various encoding, compressed (.gz, .bz2) or not
- **Excel**
- **Database**: PostgreSQL, SQL Server, Oracle DB 
- **Big Data clusters**: Spark, Hive, Impala
- **Web**
- RDS, RData
- SPSS, SAS, etc.


From sources:

- **local disc**
- **http(s)://, file://, ftp://**
- **Cloud storage services**, such as Azure Blob Storage.
