About R (programming language)

Paradigms: multi-paradigm (including functional)

Runtime: interpreted

Typing discipline: dynamic

Supported OS: Unix, Mac OS, Windows

License: Open source (GNU GPL 2)

First appeared: 1993

Stable release: 3.6.2 (2019-12-12)

Website: r-project.org

R: runtime, core packages, and IDEs

  • Multiple OSs, multiple R runtimes, multiple IDEs, terminal (console)
  • Possible to build a software stack based only on Open Source
  • Available via cloud platforms: Microsoft Azure, AWS, Google Cloud.

Install R Ecosystem

  1. Your PC host any of the following OS: Linux, Mac OS, Windows.
  2. Use CRAN or Microsoft R Open installation guide
  3. Install your favourite free IDE: RStudio, Jupiter Notebooks, or Visual Studio Code.
  4. (Optional) Install Git
  5. Start programming! :)
